# Women Safety App

## Project Description
This is a women safety mobile application developed using MIT App Inventor.
It helps users during emergency situations by sending SOS alerts and sharing location.

## Features
- SOS emergency button
- Emergency contact storage
- Live location sharing
- Alert message to multiple contacts

## Components Used
- TinyDB (Storage)
- GPS (Sensor)
- Maps
- SMS & Internet (Connectivity)
